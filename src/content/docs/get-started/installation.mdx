SoldevKit UI provides Solana-focused React components that you can add individually to your project. Each component comes with its dependencies and integrates seamlessly with Tailwind CSS.

## Installation

<Tabs items={["CLI (Recommended)", "Manual"]}>
  <Tab value="CLI (Recommended)">
    ### Prerequisites

    Before using SoldevKit UI, ensure your project has:
    - **Tailwind CSS** configured and working
    - **React** 18.x or higher
    - **TypeScript** (recommended)

    ### Add Components

    Add individual components to your project using the CLI:

    ```bash
    npx soldevkit-ui@latest add wallet-connect-button
    ```

    This will:
    - Copy the component files to your `components/ui/` directory
    - Install any component-specific dependencies (e.g., `@solana/wallet-adapter-react`)
    - Ensure proper TypeScript types are included

    ### Example Usage

    Once added, import and use components in your application:

    ```tsx showLineNumbers
    import { WalletConnectButton } from "@/components/ui/wallet-connect-button";

    export default function MyApp() {
      return (
        <div className="p-4">
          <WalletConnectButton />
        </div>
      );
    }
    ```

    ### Add More Components

    Continue adding components as needed:

    ```bash
    # Add transaction status component
    npx soldevkit-ui@latest add transaction-status

    # Add balance display component
    npx soldevkit-ui@latest add balance-display
    ```

    <Callout type="info">
      Each component is self-contained and only installs the dependencies it needs, keeping your bundle size optimized.
    </Callout>

  </Tab>
  <Tab value="Manual">
    ### 1. Configure Tailwind CSS

    Ensure your `tailwind.config.js` includes the component paths:

    ```js title="tailwind.config.js"
    /** @type {import('tailwindcss').Config} */
    module.exports = {
      content: [
        "./app/**/*.{js,ts,jsx,tsx,mdx}",
        "./pages/**/*.{js,ts,jsx,tsx,mdx}",
        "./components/**/*.{js,ts,jsx,tsx,mdx}",
        // Add this line for SoldevKit UI components
        "./components/ui/**/*.{js,ts,jsx,tsx}",
      ],
      theme: {
        extend: {},
      },
      plugins: [],
    }
    ```

    ### 2. Install Dependencies

    Install component dependencies manually:

    ```bash
    # Example for wallet components
    pnpm add @solana/web3.js @solana/wallet-adapter-react
    ```

    ### 3. Copy Component Files

    1. **Create the component directory:**
       ```bash
       mkdir -p components/ui
       ```

    2. **Copy component files** from the [SoldevKit UI repository](https://github.com/your-repo/soldevkit-ui) to your `components/ui/` directory.

  </Tab>
</Tabs>

## Usage

Once installed, import and use components in your React application:

```tsx
import { WalletConnectButton } from "@/components/ui/wallet-connect-button";
import { TransactionStatus } from "@/components/ui/transaction-status";

export default function App() {
  return (
    <div className="p-4">
      <WalletConnectButton />
      <TransactionStatus signature="your-tx-signature" />
    </div>
  );
}
```
