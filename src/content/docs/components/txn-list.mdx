## Usage

```tsx
import { TxnList } from "@/components/soldevkit-ui/transaction/txn-list";

const transactions = [
  {
    signature: "5VfYmGBjvxKjKjFxwlZQpxJGYmGBjvxKjKjFxwlZQpxJGYmGBjvxKjKjFxwlZQpxJGYmGBjvxKjKjFxwlZQpxJGY",
    block: 245678901,
    time: "2 mins ago",
    sender: "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM",
    fee: 0.000005,
  },
  // ... more transactions
];

export default function Example() {
  return <TxnList transactions={transactions} loading={false} className="border rounded-lg" />;
}
```

## Features

- **Solscan Integration**: Direct links to transaction details on Solscan
- **Loading States**: Built-in skeleton loading animation
- **Empty States**: Graceful handling when no transactions are available
- **Responsive Design**: Mobile-friendly table layout
- **Real-time Data**: Supports live transaction data with connection hooks
- **Time Formatting**: Human-readable relative time display
- **Fee Display**: Properly formatted SOL fee amounts
- **Address Truncation**: Shortened addresses for better readability

### Transaction Object

The component expects transaction objects with the following structure:

```tsx
interface Transaction {
  signature: string; // Transaction signature hash
  block: number; // Block number
  time: string; // Human-readable time (e.g., "2 mins ago")
  sender: string; // Sender's public key
  fee: number; // Transaction fee in SOL
}
```
