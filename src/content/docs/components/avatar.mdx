## Usage

```tsx
import { PublicKey } from "@solana/web3.js";
import { Avatar } from "@/components/soldevkit-ui/avatar";

const walletAddress = new PublicKey("9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM");

export default function Example() {
  return <Avatar address={walletAddress} size={48} className="border-2 border-primary" />;
}
```

## Features

- **Unique Visual Identity**: Generates consistent, unique visual identifiers for wallet addresses
- **Minidenticons**: Uses the minidenticons library for deterministic avatar generation
- **Customizable Size**: Supports any pixel size for different use cases
- **Responsive Design**: Scales appropriately across different screen sizes
- **Accessibility**: Includes proper alt text and ARIA attributes
- **Performance Optimized**: Memoized SVG generation for better performance
- **Solana Native**: Built specifically for Solana PublicKey objects

## Styling

The Avatar component uses Tailwind CSS classes and can be customized using the `className` prop:

```tsx

<Avatar className="ring-2 ring-primary ring-offset-2" />

// Hover effects
<Avatar className="transition-transform hover:scale-110" />
<Avatar className="hover:shadow-lg hover:border-primary" />
```

## Accessibility

The Avatar component includes several accessibility features:

- **Alt Text**: Automatically uses the wallet address as alt text, or accepts custom alt text
- **Semantic HTML**: Uses proper img elements with appropriate attributes
- **Keyboard Navigation**: Can be made focusable and interactive with proper event handlers

```tsx
<Avatar
  address={address}
  alt="User wallet avatar"
  className="focus:outline-none focus:ring-2 focus:ring-primary"
  tabIndex={0}
/>
```

## Technical Details

### Minidenticons

The component uses the [minidenticons](https://github.com/laurentpayot/minidenticons) library to generate deterministic SVG avatars. Each unique wallet address will always produce the same visual identifier.

### Performance

The SVG generation is memoized using `React.useMemo` to prevent unnecessary recalculations when the component re-renders with the same address.

### Data URI

The generated SVG is embedded as a data URI, eliminating the need for external image files and ensuring the avatar is always available.
