## Installation

Add components to your project using the shadcn CLI:

```bash
npx shadcn@latest add https://soldevkit.com/r/(component-name).json
```

### Examples

```bash
# Add the copy button component
npx shadcn@latest add https://soldevkit.com/r/copy-button.json

# Add the wallet connect button component
npx shadcn@latest add https://soldevkit.com/r/wallet-connect-button.json

# Using pnpm
pnpm dlx shadcn@latest add https://soldevkit.com/r/copy-button.json
pnpm dlx shadcn@latest add https://soldevkit.com/r/wallet-connect-button.json
```

That's it! The CLI will automatically:

- Copy the component files to your project
- Install required dependencies
- Configure TypeScript types
